너 재수없어 란?

재수를 막기 위한 환산점수 계산기!
작년의 데이터를 이용하여 대입을 도와줍니다!
작년 해당 과의 여러가지 정보들을 알 수 있습니다!

사용방법

1. 확인 할 과를 먼저 선택
2. 각 과목별 점수 및 등급 입력 후 점수 계산 클릭
3. 환산 점수 확인 후 결과 보기 클릭
4. 결과 확인
